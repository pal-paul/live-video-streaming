Video Management

  • POST /api/v1/videos/upload - Upload video to GCS
  • GET /api/v1/videos - List all uploaded videos
  • GET /api/v1/videos/signed-url - Generate signed URL
  • DELETE /api/v1/videos - Delete video from GCS

Broadcast Streaming

  • POST /api/v1/streams - Create new broadcast stream
  • GET /api/v1/streams - List all active streams
  • POST /api/v1/streams/:id/start - Start broadcasting
  • GET /api/v1/streams/:id/watch - Watch stream (SSE)
  • GET /api/v1/streams/:id/stats - Get stream statistics
